# mini-rag-app
this is a mini RAG app built with python and FAStapi 
it is for leaning purposes 
# requirements
 - pyhton 3.14 or highrer 
  - fastapi 
  - uvicorn 
to install the requirements run the following command 
```bash
 $ pip install -r requirements.txt 
 ```

# how to run the app to run the app run the following command 
```bash
 $ uvicorn main:app --reload 
 ```

# .env file for enviroments variables 
create a .env file in the root directory of the project respecting .env example file 
```bash 
 $ cp .env.example .env 
```
